A resposive website for travelling management, a platform that allows user to book for a visitation to a particular resort center, new relaxation center can be added by the admin and can be accessed easily by the users.

Code Link

Project Overview

This project is a travelling management system that shows travellers or visitors information about where they want to visit, how they can visit, how to make reservations, packages of visitation. it also has admin dashboard that shows number of visitors, accepted visitations (date, time, and packaged). Also a booking for visitation can be cancelled easily and admin can easily add newly discovered centers of attraction for visitation.

The Frontend was built with HTML, CSS, JavaScript and Bootstrap while Backend was done using PHP and MySQL
